## List of the **most common Windows keyboard shortcuts** 

Useful shortcuts for everyday tasks like saving, copying, pasting, undoing, and more

### üìÑ **Basic File and Text Operations**

| Shortcut   | Action     |
| ---------- | ---------- |
| `Ctrl + S` | Save       |

| `Ctrl + X` | Cut        |
| `Ctrl + V` | Paste      |
| `Ctrl + Z` | Undo       |
